ZEN CODER

Presented by Department of Computer Science and Engineering

 

a) Code Master
Level 1: Code Obfuscation (Code Swapping)

The first level is code obfuscation or code rearranging. You will be given with a jumbled code and you must rearrange the code to get the correct output. No lines can be added or removed from the given code.

Level 2: Code conversion.
The second level is code conversion. You will be given a code in one language and asked to change the code into another language. Eg:Python to Java,Java to C

General:
In both levels the code will be checked with various test cases.
All who cracks the first level will get into second level.
The prizes will be based on the speed of the contestants in the second level.
Preferred Languages: C/C++/Java

b) Dazzle Coding
Are you confident in your coding skills??? Do you try to find out the logic behind everything? Are you confident enough to code without looking at the monitor?? Then Dazzle Coding is the right event for you!!

The event is about one's logical skills and analytical skills. Though a little knowledge on programming would be helpful.



Skills: Logical, Reasoning, Knowledge on C/C++/Python

Number of rounds: 2

Round 1: Echo coding
Round 1 is all about coding and finding the logic. Contestant need to code the given programming question. After a fixed time, the other team member needs to take over the coding. The challenge is that the member will not be given the question and need to complete the coding by analysing the code and predicting the correct output.

Time: 1 hour

Round 2: Blind coding.

The contestant has to code with their monitors off by listening to code said by the team member.

Time: 1 hour




c) Sequence Hacker
Are you crazy about solving riddles and sequences? Do you try to find out the logic behind everything? Then Sequence Hacker is the right event for you!!
The event is about one's logical skills and analytical skills. Though a little knowledge on programming would be helpful. 

Skills: Logical, Reasoning, Mathematics, Knowledge on C/C++/Python 

Registration: Online 

Number of rounds: 2

Round 1: Reverse Coding
Round 1 is all about finding the logic behind the sequences and fill the missing terms. Input - output pairs will be provided and participants must decipher the link between input and output to solve the question. Or you may have to find the next term of a sequence!!!

Time : 1 hour 

Eg: Complete the sequence: 0, 1, 1, 2, 3, 5, __ You know its Fibonacci Series and the next term is 8. Pretty easy right !!!

Round 2: Pattern Mania
Contestants will be given 3 problems which accepts an input and produces an output. The problems will be either to print a sequence up to a limit or to print a pattern. You have to write a C/C++/Python program that replicates the logic behind the programs and print the desired output. 

Eg: You may asked to code the logic behind Fibonacci series. Or to print a pattern like:


*

* *


* * *



* * * *



d) Bug Tracker
Are you interested in finding errors? Do you like to know what a code says? So here is where your logic works. If you have good logic then this event will suit you.

In this event your critical thinking and logic capabilities are important. Also you should have some knowledge about C programming (syntax, working etc).



Skills: Logical, Reasoning, Critical Thinking, Knowledge on C++

Team: 4 Members

Reg Fee: 100/-
