ZEN CODER

Presented by Department of Computer Science and Engineering

 

The event that will put the geniuses behind the screen to test. A Sequence of ultimate coding events where you will have to prove your coding skills to survive each stage. Use your common sense, figure out the logic and kick the bugs out .
 
Description :
Preferred Languages: C/C++/Java/Python
 
The event will have 3 main rounds
1.Code master
Sub Rounds-
a) Code Swapping- The first level is code swapping or code obfuscation. You will be given with a jumbled code and you must rearrange the code to get the correct output. No lines can be added or removed from the given code.
b) Code conversion- The second level is code conversion. You will be given a code in one language and asked to change the code into another language.
Eg:Python to Java,Java to C
2. Sequence Hacker
Sub Rounds-
a) Reverse Coding: It is all about finding the logic behind the sequences and fill the missing terms. Input - output pairs will be provided and participants must decipher the link between input and output to solve the question. Or you may have to find the next term of a sequence!!!
Eg:
Complete the sequence: 0, 1, 1, 2, 3, 5, __ You know its Fibonacci Series and the next term is 8. Pretty easy right !!!
b) Pattern Mania: Contestants will be given a problem which accepts an input and produces an output. The problems will be either to print a sequence up to a limit or to print a pattern. You have to write a program that replicates the logic behind the programs and print the desired output.
Eg:
You may asked to code the logic behind Fibonacci series. Or to print a pattern like:
*
* *
* * *
* * * *
3. Bug Tracker
Debugging- All about finding errors or bugs and correct it.
