ZEN CODER

Presented by Department of Computer Science and Engineering

 
Description :

The event that will put the geniuses behind the screen to test. A Sequence of ultimate coding events where you will have to prove your coding skills to survive each stage. Use your common sense, figure out the logic and kick the bugs out .

 
Preferred Languages : C/C++/Java/Python
 

ROUNDS

Round 1 : Code master

Code Swapping - The first level is code swapping or code obfuscation. You will be given with a jumbled code and you must rearrange the code to get the correct output. No lines can be added or removed from the given code.

Round 2 : Sequence Hacker

Reverse Coding : It is all about finding the logic behind the sequences and fill the missing terms. Input - output pairs will be provided and participants must decipher the link between input and output to solve the question. Or you may have to find the next term of a sequence!!!
Eg:

Input  Output
 2,11 20
 4, 9 32
 7, 8 49
 4,10 ??

Code to find the missing number!!!

Round 3 : Echo Coding

The problem statement will be given to both the team members .After which the first member should code for a stipulated time and then the second member should take on the program
This process will continue iteratively with the time lapse decreasing at each iteration.

Round 4 : Bug Tracker

Debugging - All about finding errors or bugs and correct it.


 

RULES

  • The event is open to all undergraduate and postgraduate students of Engineering from any branch.
  • College ID card is mandatory.
  • Only 2 members per team is allowed.
  • Participants of different colleges can be one team.
  • Spot registration will be available if the number of teams is less than 10.
